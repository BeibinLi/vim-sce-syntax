

**vim-sce-syntax**
=================

Description
-----------------

This project adds SDL and PCL support to vim. It covers syntax and so on.

SDL and PCL languages are used in Neurobs Presentation Software. Strange languages.

Hope this syntax file could help you work with Presentation.


## Installation
### Manually
1. Put all files under $VIM.

### Vundle
1. Add the line
	
		"Plugin 'BeibinLi/vim-sce-syntax' " 

into your .vimrc file

2. Install with `:PluginInstall`


### Pathogen
1. If you prefer to install plugins using [Pathogen.vim](https://github.com/tpope/vim-pathogen).
	
		cd ~/.vim/bundle
		git clone git://github.com/BeibinLi/vim-sce-syntax

### NeoBundle (Recommended)
1. Add the following configuration to your `.vimrc`.

        NeoBundle 'BeibinLi/vim-sce-syntax'

2. Install with `:NeoBundleInstall`.
